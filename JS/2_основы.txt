// Задание: Что выведется в консоль и почему?
// const a = () => console.log("a");
// const b = () => console.log("b");
// const c = a || b ? a() : b();
// console.log(c);

Ответ: вывод будет "a" и "undefined". При выполнении выражения "a||b" идет проверка функции "a" на истинность. В этом случае "a" является истинным значением, поэтому выводится "a". После вызова a(), функция "a" не возвращает явное значение, поэтому "c" становится со значением "undefined".
